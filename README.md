# Parking Management System

## Description
A C++ based Parking Management System developed using Object Oriented Programming concepts.

## Features
- Vehicle Entry
- Vehicle Exit
- Parking Fee Calculation
- Space Availability

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Constructors
- Abstraction
